# Bruno Pelegrino — Portfolio

Portfólio profissional focado em **Power Platform**, **automações corporativas**, **integrações com APIs** e **backend**.

## 🛠 Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Lucide Icons

## 📁 Estrutura
```bash
.
├── src/
│   ├── App.tsx
│   ├── main.tsx
│   └── styles.css
├── index.html
├── package.json
├── tailwind.config.js
├── postcss.config.js
├── tsconfig.json
├── tsconfig.app.json
└── vite.config.ts
```

## 🚀 Execução local
```bash
npm install
npm run dev
```

## 🏗 Build
```bash
npm run build
npm run preview
```
