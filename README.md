# Bruno Pelegrino — Portfolio

Portfólio profissional focado em **Power Platform**, **automações corporativas**, **integrações com APIs** e **backend**.

## 🔎 Objetivo
Apresentar experiência e resultados de forma rápida para recrutadores técnicos e não técnicos, com foco em conversão para contato.

## 🛠 Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Lucide Icons

## 📁 Estrutura
```bash
.
├── src/
│   ├── App.tsx
│   ├── main.tsx
│   └── styles.css
├── index.html
├── package.json
├── tailwind.config.js
├── postcss.config.js
├── tsconfig.json
├── tsconfig.app.json
└── vite.config.ts
```

## 🚀 Execução local
```bash
npm install
npm run dev
```

## 🏗 Build
```bash
npm run build
npm run preview
```

## 🌍 Deploy recomendado
- **Vercel** (ideal para deploy contínuo com GitHub)
- **Netlify** (simples e rápido)

### Passos (Vercel)
1. Importar repositório no Vercel.
2. Framework preset: `Vite`.
3. Build command: `npm run build`.
4. Output directory: `dist`.
5. Deploy.
